Exercise 3: Paragraphs and Line Breaks

A paragraph in HTML is created with the <p> element. Every time you use <p> ... </p>, the browser starts a new block of text with extra spacing.

A line break is created with the <br> tag. It is an "empty element" — it has no closing tag. You place <br> at the end of a line when you want the following text to continue on the next line, but still inside the same paragraph.

Without line breaks

This is the first line. This is the second line. This is the last line.

Example with line breaks (notice we add <br> after each line except the last one):
This is the first line.
This is the second line.
This is the last line (no <br> needed after this).

Exercise 4: Ordered and Unordered Lists

In HTML, lists are used to group related items. There are two main types:

  • <ol> — an ordered list, where each item is automatically numbered.
  • <ul> — an unordered list, where each item is marked with a bullet point.

Example of an Ordered List

  1. First item
  2. Second item
  3. Third item

Example of an Unordered List

  • Apples
  • Bananas
  • Cherries
  • Dates

Important Note on Closing Tags

Most HTML elements come in pairs: an opening tag and a closing tag. The closing tag is written the same way as the opening tag, but it starts with a slash /.

Example: <p>This is a paragraph</p> - <p> = opening tag - </p> = closing tag (notice the /)

If you forget the / in the closing tag, the browser may guess where the element ends, but the result can be unpredictable.

Some elements, called void elements, do not have closing tags at all. Examples: <br> (line break), <img> (image), <hr> (horizontal rule), and <meta>. These are always written as a single tag without /.

Hyperlinks in HTML are created using the <a> (anchor) tag. The href attribute defines the destination URL.

Exercise 7: Tables

Tables in HTML are created using the <table> element. A table is made up of rows (<tr>), and each row contains cells. - <th> is used for a table header cell (bold, centered by default). - <td> is used for a table data cell.

Example Table (3x3)

Header 1 Header 2 Header 3
Row 1, Col 1 Row 1, Col 2 Row 1, Col 3
Row 2, Col 1 Row 2, Col 2 Row 2, Col 3

In this example: - The first row uses <th> for column headings. - The following rows use <td> for normal table cells. - The border="1" attribute adds a simple border (later we can use CSS for styling).
